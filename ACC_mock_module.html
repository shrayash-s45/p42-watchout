· Submodules 1, 3, 4 can be largely built with just (Probe42 + AOC-4 attachments) — the documents in this ACC folder.

· Several specs (filing-year regex, severity rules, clustering thresholds, EPFO avg-delay) are placeholder judgments; their tuning is on the engg follow-ups list, not a spec gap.
· No proposed analysis in the spec is structurally impossible with the sources the spec promises (P42 + AOC-4 + Watchout + downstream module feeds). What this mock surfaces is the dependency on Watchout (Submodule 2) and on per-FY auditor PDFs (Submodule 4).
